# Reproducibility Notes

All launch, smoke, and preparation commands must record:

- `R.version.string`
- `Sys.which("Rscript")`
- `R.home()`
- `.libPaths()`
- repository branch and commit
- run tag
- shared source root
- source file hashes
- full row manifest path
- stage manifest path

R 4.6.0 or newer is required. On Muscat, plain `Rscript` should resolve to:

`/data/jaguir26/local/opt/R/4.6.0/bin/Rscript`

Do not call `/usr/bin/Rscript` for this study.

The exDQLM/DQLM harness consumes the frozen shared source bundle generated by
the Q-DESN validation side. It does not generate, append, or splice sources.
