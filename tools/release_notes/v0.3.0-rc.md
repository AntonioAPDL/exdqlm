## Key conflict decisions (0.2.0 API preserved)
- Kept 0.2.0 generics/API surface; removed stale Rd files for `combineMods`/`dlmMod` (API already replaced by `generics_etc.R`).
- Preserved `get_gamma_bounds()` export (user-facing wrapper), alongside 0.2.0 exports.
- Gamma bounds now use a unified internal helper (`.gamma_bounds`) that calls the C++ bounds, validates against the defining equations (log-scale), and falls back to the R reference bounds when suspicious; applied in ISVB/MCMC with regression tests (incl. p0=0.001 and p0~0.994...).
- In `exdqlmISVB`, kept optional C++ sampling bridges but **defaulted to R** paths; posterior predictive uses `rexal()` (avoids new brms dependency).
- Makevars: OpenMP remains optional and uses SHLIB_OPENMP flags; BLAS/LAPACK linked via macros.
- Build hygiene: `.Rbuildignore` includes `.git`, `check-logs/`, `scripts/`, and `README.html` to avoid CRAN NOTES.
